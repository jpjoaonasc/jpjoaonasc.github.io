Software para cálculo de sub-redes

Implementado em Java, usando o NetBeans IDE 8.2, o software é capaz de apresentar, a partir de dados fornecidos pelo usuário, informações sobre determinado IP. O código-fonte encontra-se bem comentado e estruturado.

Basicamente, o software é composto de duas classes: uma responsável pela interface gráfica e outra pelos cálculos necessários. Em primeiro lugar, a interface gráfica foi inteiramente desenvolvida usando um GUI do próprio NetBeans, facilitando muito o processo. Cores de fundo e fontes foram alteradas para melhor apresentação.

O "coração" do software está na classe “Calculo.java”, onde são feitos todos os cálculos dos endereços de IP. Dentro da classe, encontram-se métodos responsáveis pela validação do endereço de IP e do prefixo de rede fornecidos pelo usuário. Uma série de condições dentro dos métodos “VerificaIP” e “VerificaPrefixo” garantem que nenhum endereço ou prefixo inválido siga o fluxo do programa e cause algum problema.

Logo após a verificação, os demais métodos realizam tarefas como: classificar o IP de acordo com o primeiro octeto, calcular a máscara de rede em formato decimal, descobrir o endereço de rede e broadcast e, caso for classe “C”, imprimir uma tabela com todas as sub-redes para aquele bloco de endereços.
